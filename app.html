📋 How to add a new study guide

  1. Drop your new HTML file (e.g. CIW_JavaScript_Study_Guide.html) into the guides/ folder app.html
  2. Open app.html and find the comment "add new cards below this line"
  3. Copy an existing <a class="guide-card"> block and update the href, title, description, and meta fields
  4. Change the color class (c-blue, c-green, c-purple, c-amber) and icon emoji to differentiate it
  5. Deploy to Vercel — vercel --prod or push to your connected Git repo
